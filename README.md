# About

This is a basic app to organize day to day life todos and all other tasks. Technologies used:

- NextJs
- Tailwind
- Mongodb
- Typescript

# Objectives

- to use nextjs, mongoosejs and typescript together.
- to build projects from scratch.

## How to Run this Project?

Clone this repository. And inside the cloned folder:
At first, install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, rename the .env.example to .env.local and change the required environment variables.
And finally, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Or build and start the production server:

```bash
npm run build && npm run start
# or
yarn build && yarn start
# or
pnpm build && pnpm start
# or
bun build && bun start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
